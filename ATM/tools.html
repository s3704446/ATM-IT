Github:

We will use GitHub to build dreft website for this project. GitHub is a web tool that provides hosting for software development and version control using Git. The reason we use GitHub is we are familiar to use this tool after finish Intro to IT course. GitHub is also a feel tool to store our code for our project. The reason why we don't use another web design tool like Wix (https://www.wix.com/professionalwebsite/leader-topten) is we have no experience to use this tool before, it could cause some unexpected mistakes when we create our website.

We have weekly group meeting in Sunday, every group member will join and share inspirations and what every member have done in this week. Decision-making Processes

All project decisions have been put forward with group members and our group decide if we take this decision and how we process it. Every week we have new decisions put out and some are already taken part in our project. Yuliang Lei decide most part of project decision, other group members help him figure out how to achieve this decision in project.
For example, Lei decide we need to add social feature in our project, and Chen thinks our app can ask to take location information from users and ask them if they want to share it to another user whom bought same product.
